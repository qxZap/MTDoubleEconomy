Go to steam.
Go to libary.
Right click "Motor Town Behind the Wheel"
Hover over "Manage"
Click "Browse Local Files"
a new folder shall appear
Go into "MotorTown", then "Content" then "Paks"


Chose which pak(mod) do you want to install.
If you want the full version of the mod, use the pak from the folder "AllChanghes" within the archive

If you want only specific changes, use the pak from the folder "IndividualChanges" within the archive

If you are playing with Rak's Fragile Logistics or Stack Boxes and Pallets then use the according compatibility mod from the folder "Compat"
There are 3 files:
- one for Stack Boxes and Pallets compatibility
- one for Rak's Fragile Logistics
- and one for both of the 2 mods

Drop the desired mod/mods from the archive in the "Paks" folder
